# Night 2: Security Governance & Mindset
## Project Overview
This lab establishes the strategic framework for my 2026 security environment.

## 1. Security Philosophy (NIST CSF 2.0)
My philosophy centers on the **Govern (GV)** function. I believe that 
technical security must be guided by clear policy and risk assessment.

## 2. CIA Triad Mapping
* **Confidentiality:** Managed via AES-256 encryption.
* **Integrity:** Protected through File Integrity Monitoring (FIM).
* **Availability:** Ensured via redundant backups and 99.9% uptime targets.

## 3. AAA & Non-Repudiation
* **Authentication:** Phishing-resistant MFA (CIS Control 6.3).
* **Authorization:** Role-Based Access Control (RBAC).
* **Accounting:** Immutable logging to ensure **Non-Repudiation**.
